# 🎓 Student Management System (Python)

## 📌 Description
A simple Python-based Student Management System that allows users to add student details, view results, and check pass/fail status.

## 🚀 Features
- Add student details (Name, Roll No, Marks)
- View student results
- Pass/Fail system based on marks

## 💻 Technologies Used
- Python
- Loops
- Conditional Statements
- Lists

- Basic Python programming
- Working with lists and tuples
- Building simple real-world projects
